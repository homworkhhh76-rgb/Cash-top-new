## ربط ماسح الجوال بدون إنترنت

يوجد طريقتان:

### الطريقة 1: WebRTC يدوي بدون سيرفر

من الكاشير اضغط ربط الجوال، انسخ رمز WebRTC الكبير للهاتف، ثم انسخ جواب الهاتف إلى نافذة الكاشير. بعد الاتصال، أي باركود من صفحة الماسح يضاف للسلة مباشرة.

### الطريقة 2: جسر محلي اختياري على نفس الشبكة

على الكمبيوتر داخل مجلد المشروع شغل:

```bash
node local-bridge.js
```

ثم افتح الرابط الذي يظهر في الطرفية على الكمبيوتر والجوال. هذه الطريقة لا تحتاج إنترنت، لكنها تحتاج أن يكون الجهازان على نفس شبكة Wi-Fi.

## v6 إصلاح تسجيل الدخول والمزامنة
